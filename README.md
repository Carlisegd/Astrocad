# Astrocad
Projeto feito durante o curso Full Stack da Labenu.

Link do surge:
https://dull-tiger.surge.sh/

O que funciona no projeto:
- Campo de Busca;
- Filtro por ordenação;
- Adicionar na sacola;
- Remover da Sacola;
- Filtro por valor máximo e mínimo.

Autores do projeto: Carlise, Antônio e Diego
